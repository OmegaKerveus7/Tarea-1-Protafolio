Proyecto Transversal: API Python + Compilador Java + UM++

Desarrollé un pipeline completo: API en Python que recibe instrucciones de alto nivel, las compila usando un compilador escrito en Java para mi lenguaje propio UM++, y envía los comandos de bajo nivel al Batirrover.

  • Lenguaje UM++: Tipado estático, inferencia, pattern matching
  • Compilador Java: AST, análisis semántico, generación de bytecode
  • API Python: FastAPI, WebSockets, comunicación serial
  • Integración: Pipeline automatizado IDE → Robot
